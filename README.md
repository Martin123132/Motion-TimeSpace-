# Motion-TimeSpace Research Repository

This repository contains exploratory, empirical-first, version-controlled research artifacts across Motion–TimeSpace theory, galaxy dynamics, cosmology, orbital dynamics, quantum/field work, mathematics, and supporting tools.

## Start here
- **Galaxy work**: `galaxy-work/`
- **Cosmology work**: `cosmology/`
- **Core MTS framework**: `core-mts-framework/`
- **Tools and infrastructure**: `tools-and-infrastructure/`

## Folder guide
- `galaxy-work/` — galaxy-law drafts, SPARC/rotation analyses, galaxy notebooks, experiments.
- `cosmology/` — activation-cosmology, CMB cold-spot work, Hubble tension, JWST, notebooks.
- `core-mts-framework/` — action principle, gravity, relativity, thermodynamics, field-theory foundations.
- `orbital-dynamics/` — orbital decay/hierarchy/local scaling.
- `quantum-particle-field/` — leptons/neutrinos, quarks/protons, Yang–Mills, fusion, measurement.
- `mathematics/` — Riemann/zeta, prime curvature, Navier–Stokes, stability/regularity.
- `tools-and-infrastructure/` — GMW, universe mapper, GRB kit, visualisations, computational utilities.
- `ai-ethics-and-governance/` — AI limits, liability, epistemic anchors, human impact.
- `documents/` — docx/pdf/uploaded miscellaneous files.
- `archive/` — old drafts, duplicate titles, and uncategorised legacy material.
