## 3. Parent Variation Contract

The parent action must be varied as:

```text
S_parent
  = S_EH[g/e]
  + S_matter[g/e, psi]
  + S_X[g/e, X]
  + S_projector[P_D, X, partial D, possibly g/e]
  + S_boundary[g/e|partial D, X, P_D, Y_partialD]
  + S_domain[chi_D, n_mu, L_cg, ...].
```

So the total variation is:

```text
delta S_parent
  = delta_g S
  + delta_psi S
  + delta_X S
  + delta_P S
  + delta_boundary S
  + delta_domain S.
```

Each term has to be owned.

| Term | Variation role | Local-GR requirement | Status |
|---|---|---|---|
| `S_EH` | gives `G_munu + Lambda g_munu` | survives as only local exterior metric operator | target operator |
| `S_matter` | gives `T_matter_munu` | universal coupling to one geometry | contract, not derived |
| `S_X` | gives auxiliary equations and metric stress | pure gauge/screened/boundary-only locally | open |
| `S_projector` | gives projector stress or constraint force | metric-independent local bulk or retained residual | hard gate |
| `S_boundary` | gives boundary flux/current/stress | conserved monopole only may survive | open |
| `S_domain` | gives domain/preferred-frame terms | covariant/dynamical/pure-gauge domain data | open |

This is the first important tightening:

```text
local GR is not just about E_MTS_munu.
It is also about every force created by fields used to define E_MTS_munu.
```

## 4. Ward Identity Derivation

Under a diffeomorphism generated by `xi^mu`:

```text
delta_xi g_munu = 2 nabla_(mu xi_nu),
delta_xi Phi = L_xi Phi
```

for every parent field `Phi`.

If the full parent action is diffeomorphism invariant:

```text
delta_xi S_parent = 0.
```

After integrating by parts, the identity has the structural form:

```text
nabla_mu(T_matter^munu + kappa^-1 E_MTS^munu)
  + F_X^nu
  + F_P^nu
  + F_boundary^nu
  + F_domain^nu
  + F_matter_nonmetric^nu
  = 0.
```

This is the checkpoint's main result.

The short version:

```text
conservation is true only after the force ledger closes.
```

If:

```text
F_X^nu
= F_P^nu
= F_boundary^nu
= F_domain^nu
= F_matter_nonmetric^nu
= 0
```

then:

```text
nabla_mu(T_matter^munu + kappa^-1 E_MTS^munu) = 0.
```

But if any `F` term survives, it must be retained as a physical residual.

## 6. Projector Variation Fork

| Fork | Ward result | Local-GR status | Claim allowed |

| metric-independent topological `P_D` | `F_P_bulk = 0` if covariant and constraint-owned | best route | conditional N5 support only |
