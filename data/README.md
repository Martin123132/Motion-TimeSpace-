# Data Policy

Large third-party datasets and generated run outputs are intentionally not committed to this repository.

The public repository keeps compact residual/register artifacts where they are needed for auditability. Large raw data files should be obtained from their original sources according to their own licensing and citation requirements.

Excluded by default:

- virtual environments;
- raw `runs/` directories;
- downloaded cosmology grids;
- large covariance/data files;
- cache folders;
- notebook checkpoints.
