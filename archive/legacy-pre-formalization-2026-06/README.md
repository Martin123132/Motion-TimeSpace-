# Legacy Pre-Formalization Archive

This folder preserves the older public repository material for provenance.

The current public entry point is the repository root. Files in this archive may contain older language, incomplete drafts, exploratory claims, notebooks, and rough work that predate the current claim-ceiling discipline.

Do not read archived files as the current status of the Motion-TimeSpace research programme.

Large legacy notebooks are omitted from the current tree to keep the public repository lightweight and avoid committing stale machine-output artifacts. They remain recoverable from repository history if needed.
