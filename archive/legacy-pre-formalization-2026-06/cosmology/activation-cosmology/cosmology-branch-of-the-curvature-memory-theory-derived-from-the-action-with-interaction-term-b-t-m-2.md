--------------------------------------------------------------
1. ACTION
--------------------------------------------------------------

    S = ∫ d^4x √(-g) [

            R/(16πG)
          - 1/2 g^{μν} ∂_μ M ∂_ν M
          - a M^4 / 4
          + b T M^2
          + L_b

        ]

    M(x) = curvature–memory scalar
    a    = quartic self-interaction
    b    = coupling to matter trace
    T    = T^μ_μ (baryons)
    L_b  = baryonic matter Lagrangian

For dust:

    T = -ρ_b

    b T M^2 = - b ρ_b M^2


--------------------------------------------------------------
2. FRW BACKGROUND
--------------------------------------------------------------

    ds^2 = -dt^2 + A(t)^2 δ_ij dx^i dx^j

    H = Ȧ / A

    M = M̄(t)
    ρ_b = ρ̄_b(t)

    M̈̄ + 3H Ṁ̄ + a M̄^3 + 2 b ρ̄_b M̄ = 0

    ρ̄_b(a_cos) = ρ_b0 a_cos^(-3)

3. EFFECTIVE POTENTIAL
--------------------------------------------------------------
